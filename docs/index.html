How It Works

60 seconds to live

1

Enter your domain & repo

Tell us your GoDaddy domain and GitHub repository. That's all the input we need.

gg-deploy plan mydomain.com user/repo
2

We configure everything

DNS A records point to GitHub's IPs. CNAME file added to your repo. GitHub Pages enabled. SSL provisioned automatically.

3

You're live

Your site is deployed. DNS propagates in 10-60 minutes. Check status anytime with health checks.

gg-deploy status mydomain.com user/repo
